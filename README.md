## Method (one paragraph)

`main_v31.py`: gradiometers only; subject-wise z-score; pooled ANOVA F-score
on imagine-train labels selects ~14 of 204 grad channels (bilateral
temporo-parietal); 7 ensemble members, each at a different top-N (1, 2, 3, 4,
5, 7, 10) sensor count, each running StandardScaler + (optional) PCA + L2
Logistic Regression; soft probabilities averaged; per-test-subject Hungarian
assignment enforces a uniform 5-per-class quota (3 per class for the 30-trial
test subject sub-01).

`compliant_baseline.py`: same feature extraction and Hungarian assignment,
but the classifier is trained exclusively on **localizer** epochs with
**localizer** labels, pooled across all 29 subjects. No imagine-train labels
are used for classifier supervision. This complies with line 53 of the
overview, and scores near chance (≈ 0.10–0.13).

## What did *not* work (short list)

- Cross-modal transfer (visual localizer → imagery), in any of CORAL, SRM,
  shared-latent-space, template matching, per-subject LR.
- Late imagine windows (> 0.75 s).
- Source-space reconstruction with template MRI (no individual MRI).
- Transductive whitening, ICA artifact removal, sub-window stacking,
  sample bagging, raw-MEG CNN, v31-family weighted majority vote, MILP
  leaderboard inversion with up to 35 known scores.

See `FINDINGS_summary.md` for the full audit trail.
